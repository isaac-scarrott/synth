Settled — the echo shows membership + reading order only, never geometry; a group is a bare horizontal band of tiles (no enclosing container), always horizontal, ordered top-to-bottom / left-to-right; active member keeps the .session--open accent; past ~3 members non-active tiles go icon-only (hover-expand restores name + ⋮); splits are always within one branch, so the band always sits inline under its branch row; a tile behaves like any session row — its ⋮ opens ⌘K drilled to that session, where Unsplit is a flat command beside Rename / Close (no popover — not a Synth pattern); drag a tile out is the fast alternative; creating a split stays the drag gesture (001), with a ⌘K/keyboard create command left to 007.
